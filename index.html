5-Week Rota

Shift Rota → Calendar

Set up your rotating shift pattern once, mark booked days off, then export a calendar file you can import into Outlook and share with anyone.

1Your shift types

Leave paid hours blank to use the raw start–end duration. Set it explicitly if breaks are unpaid, or for shifts with no clock times (like "AR").

2Build your 5-week pattern

Day 1 of Week 1 is whatever date you set below. Pick a shift for each day — leave as "—" for a day off in the normal pattern.

3Rota start date

The calendar date that lines up with Day 1, Week 1 above.

4Adjust specific days

Use this for booked days off, one-off changes, or swapping shifts with someone — these always override the pattern above for that exact date.

Change a single day

Book a holiday

5Pay rates

Set your hourly rate for each kind of day — week totals above update automatically.

6Generate your calendar

To use in Outlook: Calendar → Add calendar → Upload from file → select the downloaded .ics.
To share: send that same file to anyone else, or once it's in Outlook, share that calendar from there. Whenever you add a new day off, re-download and re-import to update it.

7Publish live calendar

Give this link to anyone — it refreshes with whatever you last published here. Most calendar apps re-check it roughly once a day.

8Backup & restore

If automatic saving isn't sticking, back up your full setup to a file here, and restore it any time.
