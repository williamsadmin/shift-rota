Admin Templates

Create & import

Build shift and rota templates, or import them from a PDF/text — everyone browses and adds from the Templates tab.

Import duty sheets from PDF

Upload one or more duty PDFs (like a trentbarton-style DYxx sheet). Each one is parsed into a duty board automatically and added as a shared template — review and edit before importing. Once created, any user can add it to their own rota as normal.

Import a full rota

Load a rota PDF (or paste it as text) to fill the rota template builder below automatically. Format: one line per week, seven days Sun → Sat separated by commas or tabs, each day a shift code with optional times like IB43 0737-1744 — REST or a gap for a day off.
